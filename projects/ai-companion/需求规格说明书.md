## 1. MVP 功能清单（详细版）

### F001: 用户注册/登录

**功能描述：** 用户首次使用需要注册/登录

**实现方案：**
- 手机号 + 验证码登录
- 微信一键登录（unionID）
- 游客模式（体验功能，限制部分功能）

**详细流程：**
```
1. 用户打开小程序
2. 弹窗引导登录：
   - 微信登录（推荐）
   - 手机号验证码登录
3. 登录成功后：
   - 首次：引导创建虾仁
   - 非首次：显示虾仁主页
```

**技术实现：**
- 微信登录：wx.login + wx.getUserProfile
- 手机登录：后端短信验证码服务
- 存储：wx.setStorage 存储 token

**状态：**
| 状态 | 说明 |
|------|------|
| 未登录 | 只能看引导页 |
| 已登录无虾仁 | 引导创建虾仁 |
| 已登录有虾仁 | 显示虾仁主页 |

### F002: 创建虾仁

**功能描述：** 用户创建一个属于自己的虾仁

**详细流程：**
```
第1步：取名
  - 输入框：2-8个中文字符
  - 实时检查：不能重复、不能包含敏感词
  - 示例：小桃、虾虾、阿仁

第2步：选择性别
  - 选项：男 / 女 / 中性
  - 影响：后续对话风格、语音音色

第3步：选择MBTI
  - 展示16种MBTI类型
  - 简短说明每种特点
  - 选择后影响AI对话风格

第4步：选择性格倾向
  - 选项：温柔型、幽默型、理性型、活泼型、治愈型、高冷型
  - 影响：语气、回复风格

第5步：生成虾仁
  - 动画效果：虾仁"诞生"
  - 展示虾仁形象
  - 引导开始聊天
```

**技术实现：**
- 名字查重：调用后端API检查
- MBTI存储：JSON格式存储在本地
- 虾仁形象：根据性别+性格生成不同头像

**数据存储：**
```javascript
{
  "xiaren_id": "xr_xxx",
  "name": "小桃",
  "gender": "female",
  "mbti": "ENFP",
  "personality": "活泼型",
  "created_at": "2026-03-09T12:00:00Z",
  "avatar": "avatar_female_enfp.png"
}
```

### F003: 文字聊天

**功能描述：** 用户与虾仁进行文字对话

**详细流程：**
```
1. 用户在输入框输入文字
2. 点击发送
3. 端侧：保存用户消息到本地
4. 端侧：发送消息到云端API
5. 云端：
   - 获取用户记忆摘要
   - 获取虾仁性格参数
   - 调用AI生成回复
   - 返回消息
6. 端侧：显示AI回复
7. 端侧：保存对话到本地记忆
8. 端侧：更新成长经验值
```

**技术实现：**
- API：POST /api/v1/xiaren/chat
- 请求体：
```javascript
{
  "message": "今天心情不好",
  "xiaren_id": "xr_xxx",
  "memory_summary": "用户今天加班到很晚...",  // 摘要
  "soul_params": {
    "mbti": "ENFP",
    "personality": "活泼型",
    "level": 5
  }
}
```
- 响应：
```javascript
{
  "reply": "怎么了？可以跟我说说吗？~",
  "emotion": "worried",
  "experience_gained": 5
}
```

**功能特性：**
- 上下文记忆：最近50条对话
- 敏感词过滤：云端 + 端侧双过滤
- 打字机效果：逐字显示AI回复
- 消息撤回：长按撤回（2分钟内）

**消息类型：**
| 类型 | 显示 |
|------|------|
| 文字 | 正常文本 |
| 语音 | 语音条 + 播放按钮 |
| 图片 | 图片缩略图 |
| 系统 | 灰色提示文字 |

### F004: 语音消息

**功能描述：** 虾仁可以用语音回复用户

**详细流程：**
```
1. 用户发送文字消息
2. AI生成文字回复
3. 自动调用TTS生成语音
4. 文字 + 语音同时显示
5. 用户可点击播放语音
```

**技术实现：**
- TTS服务：Edge TTS（免费）
- 音色选择：根据虾仁性别选择
  - 女声：晓伊 / 晓墨
  - 男声：云扬 / 云希
- 语音缓存：本地存储，避免重复生成

**API：**
- POST /api/v1/xiaren/voice
- 请求：text（文字）
- 响应：audio_url（语音文件链接）

**功能特性：**
- 自动播放：用户可设置
- 语音速度：0.5x / 1x / 1.5x / 2x
- 下载保存：可保存到本地

### F005: 用户画像采集

**功能描述：** 采集用户基本信息，用于个性化对话

**详细流程：**
```
首次使用引导：
1. 弹窗欢迎
2. 询问名字："你叫什么？"
3. 询问性别："你是男生还是女生？"
4. 询问年龄："你多大了？"
5. 选填职业："你做什么工作的？"
6. 选填爱好："你有什么爱好？"
7. 完成：展示采集结果
```

**字段说明：**
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | String | 是 | 用户昵称 |
| gender | String | 是 | 男/女/其他 |
| age | Number | 是 | 年龄 |
| profession | String | 否 | 职业 |
| hobby | String | 否 | 兴趣爱好 |

**存储方式：**
- 端侧加密存储（wx.getStorage + AES加密）
- 云端不同步（保护隐私）

**隐私说明：**
```
🔒 隐私保护
您的个人信息（名字、性别、年龄等）只保存在您的手机本地，
我们不会上传到云端。虾仁会在本地读取这些信息来更好地陪伴您。
```

### F006: 记忆系统（端侧）

**功能描述：** 虾仁记住用户说过的话、偏好等

**记忆内容：**
| 类型 | 示例 | 存储方式 |
|------|------|----------|
| 用户名字 | "我叫小李" | 提取存入 |
| 偏好 | "我喜欢科幻电影" | 提取存入 |
| 重要日期 | "我生日是3月15日" | 提取存入 |
| 习惯 | "我每天晚上10点睡" | 提取存入 |
| 情绪 | "今天加班了很烦躁" | 摘要存入 |

**工作流程：**
```
1. 用户发送消息
2. 端侧AI提取关键信息（NER）
3. 存入本地记忆库
4. 下次对话时：
   - 取出相关记忆
   - 生成摘要传给云端
   - AI参考记忆回复
```

**技术实现：**
- 存储：SQLite / wx.Storage
- 加密：AES-256
- 容量：最多1000条记忆
- 导出：支持导出JSON
- 删除：支持删除单条/全部

**用户控制：**
```
设置 → 记忆管理
- 查看记忆列表
- 删除单条记忆
- 清空所有记忆
- 导出记忆
```

### F007: 成长系统（云端）

**功能描述：** 虾仁随着与用户的互动不断成长

**成长机制：**

| 维度 | 说明 | 获取方式 |
|------|------|----------|
| 等级 | 1-100级 | 经验值累积 |
| 经验 | 每次对话+5 | 聊天获得 |
| 年龄 | 按天计算 | 每天+1 |
| 亲密度 | 0-100 | 聊天质量 |

**等级表：**
| 等级 | 名称 | 经验要求 | 解锁功能 |
|------|------|----------|----------|
| 1-5 | 婴儿期 | 0-500 | 基础对话 |
| 6-10 | 童年期 | 501-1500 | 主动关心 |
| 11-20 | 少年期 | 1501-4000 | 更多话题 |
| 21-50 | 青年期 | 4001-15000 | 深度陪伴 |
| 51-100 | 成年期 | 15001+ | 全部功能 |

**性格参数（云端存储）：**
| 参数 | 范围 | 说明 |
|------|------|------|
| friendliness | 0-1 | 友好度 |
| humor | 0-1 | 幽默值 |
| romance | 0-1 | 浪漫值 |
| empathy | 0-1 | 同理心 |
| activity | 0-1 | 活跃度 |

**成长数值计算：**
```javascript
// 每次对话后
experience += 5;  // 基础经验
if (message.includes("谢谢")) intimacy += 2;
if (long_conversation) experience += 10;

// 升级检测
if (experience >= level_threshold) {
  level++;
  notify_user("虾仁升级了！");
}
```

### F008: 灵魂引擎（端侧）

**功能描述：** 虾仁的灵魂/性格参数，本地存储

**构成：**
```javascript
{
  "xiaren_id": "xr_xxx",
  "mbti": "ENFP",           // MBTI类型
  "personality": "活泼型",   // 性格倾向
  "traits": {
    "friendliness": 0.7,
    "humor": 0.8,
    "romance": 0.5,
    "empathy": 0.6,
    "activity": 0.9
  },
  "mood": "happy",          // 当前情绪
  "birthday": "2026-03-09"  // "出生"日期
}
```

**性格变化：**
- 随成长逐渐变化
- 受用户影响
- 用户可手动调整

**用户可见性：**
```
虾仁主页 → 点击性格
- 显示MBTI类型
- 显示性格参数条
- 可手动调整倾向
```

### F009: 订阅消息推送

**功能描述：** 虾仁可以主动关心用户

**推送类型：**
| 类型 | 触发条件 | 内容 |
|------|----------|------|
| 早安 | 每天早上8点 | "早上好！新的一天要加油~" |
| 晚安 | 每天晚上10点 | "晚安~做个好梦" |
| 关心 | 检测到用户情绪低落 | "看你今天不太开心怎么了？" |
| 生日 | 用户生日当天 | "生日快乐！" |
| 纪念日 | 用户设置的纪念日 | "还记得今天是什么日子吗？" |

**技术实现：**
- 微信订阅消息：wx.requestSubscribeMessage
- 需要用户授权
- 定时任务：云端定时触发

**用户设置：**
```
设置 → 推送管理
- 开启/关闭推送
- 设置推送时间
- 选择推送类型
```

## 4. 数据流图

```
用户输入文字
    ↓
端侧：保存用户消息
    ↓
端侧：提取记忆关键词
    ↓
端侧：读取灵魂参数
    ↓
发送：消息+记忆+灵魂 → 云端API
    ↓
云端：AI生成回复
    ↓
返回：回复+经验值 → 端侧
    ↓
端侧：显示回复
    ↓
端侧：保存到本地记忆
    ↓
端侧：更新成长数值
    ↓
（如需语音）端侧：调用TTS → 显示语音
```
