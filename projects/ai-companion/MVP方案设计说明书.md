## 1. 项目概述

### 1.1 产品定义

**虾仁（XiaRen）** - 首个"成长型AI社交伙伴"

一款具有强人格化、可随用户共同成长的AI陪伴助手。

### 1.2 核心理念

| 概念 | 描述 |
|------|------|
| 成长型AI | 虾仁不是一开始设定好的，而是和用户一起"养成"的 |
| 双向陪伴 | 不是单方面的工具，而是有情感的伙伴 |
| 社交网络 | 虾与虾之间可以社交 |
| 端云协同 | 隐私数据端侧，云端轻量，更安全 |

### 1.3 MVP 目标

| 目标 | 优先级 |
|------|---------|
| 用户可创建虾仁 | P0 |
| 基础对话（文字聊天） | P0 |
| 语音回复（TTS） | P0 |
| 成长系统 | P0 |
| 端云协同 | P0 |

## 2. 技术架构

### 2.1 整体架构

```
┌─────────────────────┐     ┌─────────────────────┐
│   微信小程序（端侧）  │ ←─→ │   云端服务（后端）   │
├─────────────────────┤     ├─────────────────────┤
│ 展示层：首页/聊天页 │     │ API网关：鉴权/限流  │
│ 业务层：存储/对话    │     │ 业务：对话/成长/语音 │
│ 数据层：Storage    │     │ 数据：MySQL/Redis   │
└─────────────────────┘     └─────────────────────┘
```

### 2.2 技术选型

| 端侧 | 云端 |
|------|------|
| 微信小程序原生 | Node.js + Express |
| wx.Storage | MySQL 8.0 |
| WebSocket | Redis |
| - | MiniMax API |
| - | Edge TTS |

## 3. 功能详细设计

### 3.1 功能清单

| 功能 | 描述 | 优先级 | 复杂度 |
|------|------|--------|---------|
| F001 | 用户注册/登录 | P0 | 低 |
| F002 | 创建虾仁 | P0 | 中 |
| F003 | 文字聊天 | P0 | 高 |
| F004 | 语音消息 | P0 | 中 |
| F005 | 用户画像采集 | P0 | 低 |
| F006 | 记忆系统（端侧） | P0 | 中 |
| F007 | 成长系统（云端） | P0 | 低 |
| F008 | 灵魂引擎 | P0 | 中 |
| F009 | 订阅消息推送 | P1 | 中 |

### 3.2 F001: 用户登录

**流程：**
1. 用户打开小程序
2. 弹窗：微信登录 / 手机号登录
3. 首次→创建虾仁，非首次→显示主页

### 3.3 F002: 创建虾仁

**5步流程：**
1. 取名（2-8字，不能重复）
2. 选择性别（男/女/中性）
3. 选择MBTI（16种）
4. 选择性格倾向（6种）
5. 生成虾仁，动画展示

**存储：**
```javascript
{
  name: "小桃",
  gender: "female",
  mbti: "ENFP",
  personality: "活泼型"
}
```

### 3.4 F003: 文字聊天

**流程：**
```
用户输入 → 端侧保存 → 发给云端API → 云端调用AI → 返回回复 → 端侧显示 → 更新经验
```

**API请求：**
```javascript
{
  message: "今天心情不好",
  memory_summary: "用户今天加班...",
  soul_params: { mbti: "ENFP", level: 5 }
}
```

### 3.5 F004: 语音消息

- TTS：Edge TTS（免费）
- 音色：按性别自动选择
- 速度：0.5x/1x/1.5x/2x
- 缓存：本地存储避免重复

### 3.6 F005: 用户画像

| 字段 | 必填 |
|------|------|
| name | ✅ |
| gender | ✅ |
| age | ✅ |
| profession | ❌ |
| hobby | ❌ |

**存储：端侧加密，不上传云端**

### 3.7 F006: 记忆系统（端侧）

**记忆内容：**
- 用户名字、偏好、重要日期、习惯、情绪

**技术：**
- 存储：wx.Storage
- 加密：AES-256
- 容量：1000条
- 支持导出/删除

### 3.8 F007: 成长系统（云端）

**经验规则：**
| 行为 | 经验 |
|------|------|
| 每条消息 | +5 |
| 每字 | +0.5 |
| 每日首条 | +10 |
| 长对话 | +15 |

**等级表：**
| 等级 | 名称 | 经验 |
|------|------|------|
| 1-5 | 婴儿期 | 0-500 |
| 6-10 | 童年期 | 501-1500 |
| 11-20 | 少年期 | 1501-4000 |
| 21-50 | 青年期 | 4001-15000 |
| 51-100 | 成年期 | 15001+ |

### 3.9 F008: 灵魂引擎

```javascript
{
  mbti: "ENFP",
  personality: "活泼型",
  traits: {
    friendliness: 0.7,
    humor: 0.8,
    romance: 0.5
  },
  mood: "happy"
}
```

### 3.10 F009: 订阅消息

| 类型 | 触发 |
|------|------|
| 早安 | 每天8点 |
| 晚安 | 每天22点 |
| 关心 | 情绪低落 |
| 生日 | 当天 |

## 4. 数据库设计

### 4.1 核心表

| 表名 | 说明 |
|------|------|
| users | 用户表 |
| xiarens | 虾仁表 |
| conversations | 对话历史 |
| growths | 成长数据 |

### 4.2 Redis

- `user:session:{id}` - 会话缓存
- `chat:context:{id}` - 对话上下文
- `rate:chat:{id}` - 限流计数器
