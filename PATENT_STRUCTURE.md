# 📜 真实专利文档结构参考

## 发明/实用新型专利结构

### 1. 申请文件
- 发明专利请求书
- 说明书
- 权利要求书
- 说明书摘要
- 说明书附图（如果有）
- 摘要附图（如果有）

### 2. 说明书结构
```
1. 技术领域
   - 本发明属于...领域
   - 具体涉及...

2. 背景技术
   - 现有技术状况
   - 存在的问题（缺点）
   - 本领域技术人员需要解决的技术问题

3. 发明内容
   - 要解决的技术问题
   - 技术方案（核心！）
   - 有益效果

4. 附图说明
   - 图1：...
   - 图2：...

5. 具体实施方式
   - 实施例1
   - 实施例2
   - 实施例3
```

### 3. 权利要求书结构
```
1. 独立权利要求
   - 前序部分：与现有技术共有的特征
   - 特征部分：本发明的特有特征

2. 从属权利要求
   - 引用在前权利要求
   - 进一步限定
```

### 4. 专利类型详细说明

#### 发明专利
- 技术含量最高
- 需要实质性特点
- 审查周期长（1-3年）
- 保护期20年

#### 实用新型
- 形状、构造改进
- 审查周期短（6-12个月）
- 保护期10年

#### 外观设计
- 产品外观设计
- 保护期15年

## 多人协同场景

### 1. 创意阶段
- A 提出发明构思
- B 分析技术可行性
- C 调研现有专利

### 2. 撰写阶段
- A 负责技术方案
- B 撰写权利要求书
- C 撰写说明书

### 3. 审核阶段
- A 审核技术内容
- B 审核法律合规
- C 审核格式规范

## AI 增强功能

1. **智能拆分** - 自动识别独立权利要求和从属权利要求
2. **查重检测** - 与现有专利库比对

4. **逻辑检查** - 权利要求之间的引用关系
5. **格式规范** - 自动按照国标格式排版
6. **多人协作** - 实时协同编辑
7. **版本管理** - 历史版本对比
8. **导出功能** - PDF/Word 格式导出
