## 1. 产品定位与愿景

### 1.1 产品愿景

> **让每一个灵机一动的想法，都能变成真正可实现的东西。**

IDEA Builder 是一个 AI 驱动的创意实现平台，致力于帮助用户将脑海中的想法快速转化为实际可用的成果，包括但不限于：专业的 AI 提示词、符合标准的专利申请文件、商业计划书、营销文案等。

### 1.2 产品定位

| 维度 | 描述 |
|------|------|
| **产品类型** | AI 创意内容生成平台 |
| **核心能力** | 自然语言输入 → 结构化输出（提示词/专利/文案） |
| **目标用户** | 创业者、技术人员、设计师、内容创作者、发明家 |
| **核心竞争力** | 零门槛、高效率、专业输出 |

## 3. 产品功能矩阵

### 3.1 核心产品线

#### 3.1.1 产品一：AI 提示词生成器 🎯

**目标**：帮助用户生成高质量的 AI 提示词，用于各种 AI 工具（ChatGPT、Midjourney、Stable Diffusion 等）

| 功能模块 | 功能描述 | 优先级 |
|----------|----------|--------|
| **提示词模板库** | 内置常用提示词模板（写作、编程、绘画等） | P0 |
| **智能提示词生成** | 用户描述需求，AI 生成优化后的提示词 | P0 |
| **提示词优化** | 已有提示词进行优化、扩展、润色 | P1 |
| **角色设定** | 自定义 AI 角色/人格，获得更精准的回复 | P1 |
| **提示词收藏** | 收藏常用提示词，方便复用 | P2 |
| **提示词社区** | 用户分享提示词，互相借鉴 | P2 |

**使用示例：**
```
用户输入：
"我想让 ChatGPT 帮我写一封求职邮件，应聘产品经理岗位"

AI 生成优化后的提示词：
"
你是一位资深HR专家，具有10年以上招聘经验。...

请帮我写一封求职邮件，应聘产品经理岗位。

要求：
1. 结构清晰，包含自我介绍、求职动机、核心优势、期待
2. 语言专业但有温度
3. 突出产品经理岗位相关的经验

请用中文回复。"
```

#### 3.1.2 产品二：AI 专利生成器 📔

**目标**：让专利申请像写文章一样简单

基于现有 V2.0 规格（详见 SPEC.md），持续优化：

| 功能模块 | 功能描述 | 优先级 |
|----------|----------|--------|
| **创意生成** | 输入技术领域，AI 生成专利创意 | P0 |
| **步骤式撰写向导** | 按专利局标准结构分步骤引导撰写 | P0 |
| **智能权利要求拆分** | 自动拆分独立/从属权利要求 | P0 |
| **专利查重** | 相似度检测与规避建议 | P1 |
| **专利优化** | 质量评分、问题诊断、自动改写 | P1 |
| **版本管理** | 自动版本快照、对比、回滚 | P1 |
| **格式规范化** | 国标格式自动检测 | P2 |
| **多人协作** | 模拟多人协同编辑 | P2 |

#### 3.1.3 产品三：AI 创意工具箱 ✨

**目标**：扩展更多创意内容生成能力

| 功能模块 | 功能描述 | 优先级 |
|----------|----------|--------|
| **短视频脚本生成** | 输入主题，生成短视频脚本（口播/种草/测评） | P1 |
| **商业计划书生成** | 输入商业模式，生成 BP 大纲/内容 | P2 |
| **营销文案生成** | 生成产品卖点、推广文案、社交媒体内容 | P2 |
| **技术文档生成** | 生成 API 文档、技术方案、技术博客 | P2 |
| **简历生成** | 输入工作经历，生成专业简历 | P2 |

```

         ↓

         ↓
```

```typescript
interface Content {

  updatedAt: Date;
```
