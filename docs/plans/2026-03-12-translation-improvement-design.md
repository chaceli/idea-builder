# 英文翻译改进设计

## 概述

IDEA Builder 是一个创意孵化工具，帮助用户将"灵感火花"变成可落地的成果。本次改进旨在优化中英文翻译，使其更准确、专业、有吸引力。

## 核心定位

**产品定位：** AI 驱动的创意孵化器

**三大核心功能：**
1. 专利申请 - 想法 → 完整专利文档
2. 项目蓝图 - 想法 → 可执行的项目方案
3. 提示词模板 - 想法 → 其他AI平台可用的prompt
