## 中文

### 项目简介

IDEA Builder 是一款 **AI 驱动的创意孵化器**，帮助用户将"灵光一现"的想法转化为可落地的成果。通过多轮对话和交互，一步步引导用户把创意变成具体的、可执行的内容。

**三大核心功能：**

1. **AI 专利生成** - 将创意想法转化为完整的专利文档，支持发明专利、实用新型、外观设计三种类型
2. **AI 蓝图生成** - 将创意想法梳理成完整的、可执行的项目蓝图
3. **AI 提示词生成** - 生成高质量的提示词模板，可在其他 AI 平台生成图片、视频等

### 核心功能

#### 🎯 AI 提示词生成
智能分析你的想法，生成高质量的提示词模板，可在其他 AI 平台生成图片、视频等。

#### 📔 AI 蓝图生成
一键生成完整的项目蓝图，帮你把创意想法转化为可落地的执行方案。

#### ✨ AI 专利生成
专业专利文档生成，支持三种专利类型：
- **发明专利** (Invention Patent) - 保护期 20 年
- **实用新型** (Utility Model) - 保护期 10 年
- **外观设计** (Design Patent) - 保护期 15 年

### 功能特性

- 📝 项目创建与管理
- 🌓 主题切换（明暗模式）
- 📱 响应式设计（移动端、平板、桌面）

- 🌐 中英文双语支持

### 技术架构

```
idea-builder/
├── index.html          # 主页面
├── css/
│   └── styles.css      # 样式文件（CSS变量、主题切换）
├── js/
│   ├── main.js         # 应用入口
│   ├── config.js       # 配置常量和翻译数据
│   ├── storage.js      # localStorage 操作封装
│   ├── theme.js        # 主题切换逻辑
│   ├── i18n.js         # 国际化模块
│   ├── projects.js     # 项目管理模块
│   ├── ai.js           # AI 生成模块
│   └── ui.js           # UI 交互模块
├── assets/
│   └── favicon.svg     # 网站图标
└── docs/
    └── plans/          # 设计文档
```

**技术栈：**
- HTML5 + CSS3 + Vanilla JavaScript (ES Modules)
- localStorage 本地存储
- 响应式布局 (Flexbox + Grid)
- CSS Variables（主题切换）
- MiniMax AI API

### About

IDEA Builder is an **AI-powered creative incubator** that transforms your "spark of inspiration" into reality. Through multi-turn conversations and interactions, it guides users step-by-step to turn creative ideas into concrete, actionable outcomes.

**Three Core Features:**

1. **AI Patent Generator** - Transform creative ideas into complete patent documents, supporting Invention Patents, Utility Models, and Design Patents
2. **AI Blueprint Generator** - Turn creative ideas into complete, executable project blueprints
3. **AI Prompt Template Generator** - Generate high-quality prompt templates for use on other AI platforms to create images, videos, and more

### Core Features

#### 🎯 AI Prompt Template Generator
Intelligently analyze your ideas and generate high-quality prompt templates for images, videos, and more on other AI platforms.

#### ✨ AI Patent Generator
Professional patent document generation supporting three patent types:
- **Invention Patent** - 20-year protection
