# IDEA Builder - 创意生成器

让每一个灵机一动的想法都能变成真正可实现的东西。

[在线演示](https://chaceli.github.io/idea-builder/) · [GitHub 仓库](https://github.com/chaceli/idea-builder)

## 🎯 简介

IDEA Builder 是一款 AI 驱动的创意生成器，帮助你将创意想法快速转化为可实现的项目方案。

## ✨ 核心功能

### AI 提示词生成
智能分析你的想法，生成高质量的 AI 提示词，让 AI 更好地理解你的需求，提升交互效率。

### AI 项目生成
一键生成符合规范的项目方案，帮助你实现创意。

### AI 创意工具
更多创意工具持续更新，助力你的创新之旅。

## 🚀 功能特性

- **项目创建** - 支持创建多种类型的创意项目
- **项目类型** - 发明项目、实用项目、创意设计
- **项目管理** - 创建、编辑、删除项目
- **主题切换** - 支持明暗模式，一键切换
- **响应式设计** - 完美支持手机、平板、电脑
- **本地存储** - 数据保存在浏览器本地，保护隐私

## 📱 界面预览

- 🎨 现代简洁的 UI 设计
- 🌙 明暗主题切换
- 📱 移动端优化
- ♿ 无障碍支持

## 🛠 技术栈

- HTML5 + CSS3 + Vanilla JavaScript
- localStorage 本地存储
- 响应式布局 (Flexbox + Grid)
- CSS 变量主题系统

## 📖 使用流程

1. 输入你的想法
2. AI 优化完善
3. 生成项目方案
4. 实现创意
