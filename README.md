# 💡 IDEA Builder - 专利生成器

> 每个人都能轻松生成专利的 AI 平台

## 🚀 愿景

让专利申请像写文章一样简单！

## 🎯 核心功能

1. **AI 专利构思** - 输入领域，AI 自动生成专利想法
2. **专利撰写** - 自动生成专利申请书、权利要求书
3. **专利查重** - 检测相似专利
4. **专利优化** - 提升专利质量和通过率

## 🏗️ 技术架构

### 前端 (Vue 3)
- Vue 3 + TypeScript
- Vite 构建
- Pinia 状态管理
- Vue Router

### 后端 (Node.js)
- Express + Koa
- MySQL + Redis
- MiniMax API

## 📱 用户流程

```
1. 输入 API Key
2. 选择专利类型（发明/实用新型/外观设计）
3. 描述发明创意
4. AI 生成专利
5. 查看/编辑/导出
```

## 🎨 页面结构

- 首页 - 功能介绍 + 开始使用
- 配置页 - 输入 API Key
- 创意生成页 - 输入想法，AI 生成专利
- 专利编辑器 - 查看/修改生成的专利
- 我的专利 - 专利列表管理
