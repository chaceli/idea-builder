## English

### About

Turn every great idea into reality. IDEA Builder is an AI-powered creative idea generator that helps transform your ideas into actionable project plans.

[Live Demo](https://chaceli.github.io/idea-builder/) · [GitHub](https://github.com/chaceli/idea-builder)

### Features

- **AI Prompt Generation** - Intelligently analyze your ideas and generate high-quality AI prompts
- **AI Project Generation** - Generate standardized project plans with one click
- **AI Creative Tools** - Continuous updates of creative tools

### Capabilities

- Create and manage creative projects
- Multiple project types (Invention, Utility, Creative Design)
- Theme switching (Light/Dark mode)
- Responsive design for mobile, tablet, and desktop
- Local storage for data privacy

### Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript
- localStorage
- Responsive Layout (Flexbox + Grid)
- CSS Variables
