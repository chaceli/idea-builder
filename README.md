# IDEA Builder - 创意生成器

让每一个灵机一动的想法都能变成真正可实现的东西。

## 特性

- 🎯 **AI 提示词生成** - 智能分析你的想法，生成高质量的 AI 提示词
- 📔 **AI 项目生成** - 一键生成符合规范的项目方案
- ✨ **AI 创意工具** - 更多创意工具持续更新

## 功能

- 创建和管理创意项目
- 项目分类（发明项目、实用项目、创意设计）
- 主题切换（明暗模式）
- 响应式设计，支持移动端

## 技术

- 纯 HTML/CSS/JavaScript
- localStorage 本地存储
- 响应式布局
